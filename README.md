# Tower
### Développé par Fabien Aubret, [SimpleDuino](http://simple-duino.com), pour [NativGaming](http://nativgaming.com)
### Spigot 1.8.8

## Description
Tower implémente le célèbre mode de jeu Tower disponible en vanilla sur un serveur, en supprimant l'utilisation de commands blocks, remplaçant l'ensemble par un unique plugin

### Dépendances:
- spigot
- Vault (?)
